## What are Slash Commands?

> "Slash Commands are customized, carefully refined prompts that control Claude's behavior in order to perform a specific task"

Slash commands are powerful tools that enable you to extend Claude Code's functionality with custom, project-specific behaviors. By creating slash commands, you can automate common workflows, enforce project standards, and streamline your development process.

## Getting Started

To use these slash commands in your project:

1. Browse the category pages to find commands relevant to your workflow
2. Click through to the GitHub repository for the command
3. Copy the command file to your project's `.claude/commands/` directory
4. Customize the command to match your project's needs
5. Invoke the command in Claude Code using `/command-name`
