Status lines are configurations and customizations for Claude Code's status bar functionality. They enhance your terminal experience by displaying important information like model details, git status, token usage, and other metrics.

## Key Features

Most status line tools offer:

- **Real-time Metrics**: Display current token usage, model information, and session statistics
- **Git Integration**: Show current branch, commit status, and repository information
- **Customization**: Support for themes, colors, and layout configurations
- **Performance Tracking**: Monitor usage patterns, costs, and burn rates
- **Visual Indicators**: Progress bars, icons, and status symbols for quick reference

## Getting Started

To use a status line tool:

1. Choose a status line implementation that fits your needs
2. Follow the installation instructions in the respective repository
3. Configure Claude Code to use your chosen status line
4. Customize the appearance and displayed metrics to your preference

<Tip>
Many status line tools are written in high-performance languages like Rust to minimize overhead and ensure your Claude Code sessions remain responsive.
</Tip>
