# Latest Additions

Discover the newest resources added to the Awesome Claude Code collection. These cutting-edge tools and skills represent the latest innovations in the Claude Code ecosystem.

## Recent Resources

<CardGroup cols={1}>
  <Card title="Claude Scientific Skills" icon="microscope" href="https://github.com/K-Dense-AI/claude-scientific-skills">
    **By K-Dense**
    
    A set of ready-to-use Agent Skills for research, science, engineering, analysis, finance and writing. According to the authors: "That's their description - modest, simple. That's how you can tell this is really one of the best skills repos on GitHub."
    
    If you've ever thought about getting a PhD... just read all of these documents instead. Features comprehensive research methodologies, analytical frameworks, and appears to function as an AI agent itself.
  </Card>

  <Card title="parry" icon="shield" href="https://github.com/vaporif/parry">
    **By Dmytro Onypko**
    
    Prompt injection scanner for Claude Code hooks. Scans tool inputs and outputs for injection attacks, secrets, and data exfiltration attempts.
    
    **Note**: Currently in early development phase but worth monitoring for security-conscious developers.
  </Card>

  <Card title="Dippy" icon="check-circle" href="https://github.com/ldayton/Dippy">
    **By Lily Dayton**
    
    Auto-approve safe bash commands using AST-based parsing, while prompting for destructive operations. Solves permission fatigue without disabling safety features.
    
    Supports Claude Code, Gemini CLI, and Cursor. Uses abstract syntax tree analysis to intelligently distinguish between safe and potentially dangerous commands.
  </Card>

  <Card title="sudocode" icon="diagram-project" href="https://github.com/sudocode-ai/sudocode">
    **By ssh-randy**
    
    Lightweight agent orchestration dev tool that lives in your repository. Integrates with various specification frameworks.
    
    As described: "It's giving Jira" - provides project management capabilities with AI agent orchestration.
  </Card>

  <Card title="claude-tmux" icon="terminal" href="https://github.com/nielsgroen/claude-tmux">
    **By Niels Groeneveld**
    
    Manage Claude Code within tmux. Features a tmux popup displaying all your Claude Code instances, enabling:
    - Quick switching between sessions
    - Status monitoring
    - Session lifecycle management
    - Git worktree support
    - Pull request integration
    
    Perfect for developers who live in the terminal and work with multiple Claude sessions.
  </Card>

  <Card title="claude-esp" icon="eye" href="https://github.com/phiat/claude-esp">
    **By phiat**
    
    Go-based TUI that streams Claude Code's hidden output to a separate terminal. Watch what Claude is really thinking:
    - View thinking processes
    - Monitor tool calls
    - Track subagent activity
    - Watch multiple sessions simultaneously
    - Filter by content type
    - Track background tasks
    
    Ideal for debugging or understanding Claude's decision-making process under the hood without interrupting your main session.
  </Card>
</CardGroup>

These latest additions showcase the evolving Claude Code ecosystem:

**Developer Experience**: `claude-tmux` and `claude-esp` improve workflow efficiency and debugging capabilities.

**Orchestration**: `sudocode` demonstrates the growing need for multi-agent coordination.
