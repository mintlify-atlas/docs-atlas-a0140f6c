General tools enhance your Claude Code workflow with session management, templates, CLI utilities, and comprehensive development frameworks.

## Session Management

<CardGroup cols={2}>
  <Card title="cc-sessions" icon="folder-open" href="https://github.com/GWUDCAP/cc-sessions">
    An opinionated approach to productive development with Claude Code by toastdev
  </Card>
  
  <Card title="Claude Session Restore" icon="clock-rotate-left" href="https://github.com/ZENG3LD/claude-session-restore">
    Efficiently restore context from previous sessions by analyzing session files and git history. Handles files up to 2GB
  </Card>
  
  <Card title="claude-code-tools" icon="toolbox" href="https://github.com/pchalasani/claude-code-tools">
    Session continuity toolkit with skills to avoid compaction and recover context. Fast Rust/Tantivy-powered full-text search
  </Card>
  
  <Card title="recall" icon="magnifying-glass" href="https://github.com/zippoxer/recall">
    Full-text search your Claude Code sessions. Run `recall` in terminal, type to search, Enter to resume
  </Card>
  
  <Card title="cchistory" icon="terminal" href="https://github.com/eckardt/cchistory">
    Like the shell history command but for Claude Code sessions. List all Bash commands Claude ran in a session
  </Card>
</CardGroup>
